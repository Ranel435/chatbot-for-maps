# MapBot - Концепция

## Что это

Backend-сервис для построения туристических маршрутов по историческим местам Москвы и МО.

## Проблема

Туристы хотят посещать исторические места, но:
- Сложно найти все интересные объекты в округе
- Нет удобного способа построить оптимальный маршрут
- Информация разрозненна по разным источникам

## Решение

Чат-бот, который:
1. Понимает запросы на естественном языке ("хочу посмотреть старые церкви")
2. Находит релевантные POI с помощью семантического поиска
3. Строит оптимальный маршрут через OSRM

## Целевая аудитория

- Туристы
- Любители истории
- Краеведы
- Организаторы экскурсий

## Ключевые метрики, которые мы хотим достичь на бекенде на старте

- RPS: 50,000+
- Latency: <100ms (p95)
- Покрытие: Москва + МО
